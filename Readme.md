# Ollama Crash Course

## Installation

Before getting started, install Ollama from the official website:
- [https://ollama.com/](https://ollama.com/)

## Getting Started

### Verify Installation

To check whether Ollama is installed and view the version number, run:

```bash
ollama -v
```

### Pull a Model

To download a model from the Ollama repository:

```bash
ollama pull <model_name>
```

### Run a Model

To start an interactive session with a model:

```bash
ollama run <model_name>
```

### Exit the Model

To exit the current Ollama model session:

```
/bye
```
